# **🏦 C# .NET BANKING SYSTEM**
A console-based banking system developed in C# and .NET to practice object-oriented programming, software design, and software engineering principles.
The project is designed to evolve over multiple versions, gradually introducing persistence, databases, APIs, and additional banking features.

## 🎯 Objectives
- Practice C#
- Practice OOP
- Practice Git
- Practice software archtecture
- Practice clean code
- Practice progressive software evolution
- Practice Data structure
- Practice Database
- Practice API

## 🛠️ Features
### Current features
  - Register
  - Login
  - Deposit
  - Withdraw
  - Transger
  - Check Balance
